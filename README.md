## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/credence.git
   cd credence
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:5173
   ```

### Running the Backend (Optional)
If you want AI features to work fully:

```bash
cd backend
npm install
npm start
```

The backend runs on `http://localhost:3001`.

### ğŸš€ Deploy to Vercel

This project is optimized for Vercel deployment:

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Ready for deployment"
   git push origin main
   ```

2. **Import to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repository
   - Framework preset will be auto-detected as **Vite**
   - Click "Deploy"

3. **That's it!** Your app will be live at `https://your-project.vercel.app`

> **Note:** The `vercel.json` file is already configured for SPA routing and security headers.
